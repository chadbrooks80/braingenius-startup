# LessonCompleteWindow

Reusable lesson summary registered as `"lesson-complete"`.

The active module supplies the title, message, and labeled stat values. The component has no vocabulary wording or fixed stat labels, so math, science, reading, and vocabulary modules can share it. It intentionally has no illustration, trophy, confetti, animation, reward copy, or progression action.

Sources: `src/components/LearningWindows/LessonComplete/` and `src/learning-modules/vocabulary/screens/lessonCompleteScreen.ts`.
